 Sistema de Gerenciamento de Alunos (SIGA)

Este projeto é um exemplo simples em JavaScript que demonstra o uso de Padrões de Projeto como:

 Prototype (Clonagem de objetos)
 Singleton (Instância única para gerenciamento)

O sistema permite criar alunos a partir de um modelo base (protótipo) e gerenciá-los através de uma única instância central.

 Tecnologias utilizadas
JavaScript (ES6+)
Programação Orientada a Objetos (POO)
 Funcionalidades
Criar um aluno base (protótipo)
Clonar alunos a partir desse modelo
Gerenciar alunos com uma única instância (Singleton)
Listar alunos cadastrados
